# mol-infer/Multimodel

このフォルダーには, 複数の所望の物性値を持つ化合物を推論するコードとインスタンスが含まれています．

フォルダー構造は以下のようになっています．
1. モジュール 3
    - 逆問題のためのMILPの計算
    - 複数の物性に対する目標値を持つ化合物の推論

詳細については、各フォルダを参照してください。
